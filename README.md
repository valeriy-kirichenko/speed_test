# 🚀 Speed Test Script

Скрипт для измерения скорости интернет-соединения путем последовательной загрузки файла с заданного URL.

## 📋 Описание

Скрипт выполняет серию HTTP-запросов к указанному URL, загружает файл порциями (чанками), замеряет время каждого запроса и вычисляет:
- Скорость каждого отдельного запроса в Мбит/с
- Среднее время запроса
- Средний объем загруженных данных
- Среднюю скорость соединения

## ⚙️ Требования

- Python 3.6+
- Библиотека `requests`

## 📦 Установка

1. Клонируйте или скопируйте скрипт в рабочую директорию:
```bash
mkdir -p ~/my_projects/speed_test
cd ~/my_projects/speed_test
```

2. Установите зависимости:
```bash
pip install requests
```

Или создайте `requirements.txt`:
```bash
echo "requests>=2.28.0" > requirements.txt
pip install -r requirements.txt
```

## 🚀 Использование

### Запуск с URL по умолчанию
Используется тестовый файл `http://speedtest.tele2.net/10MB.zip`:
```bash
python speed_test.py
```

### Запуск с произвольным URL
```bash
python speed_test.py <URL>
```

### Примеры
```bash
# Тестовый файл 10 МБ
python speed_test.py http://speedtest.tele2.net/10MB.zip

# Собственный URL
python speed_test.py https://example.com/large-file.zip
```

## 🔧 Настройка

Параметры можно изменить непосредственно в коде:

| Параметр | Значение по умолчанию | Описание |
|----------|----------------------|----------|
| `num_requests` | `10` | Количество последовательных запросов |
| `chunk_size` | `8192` | Размер чанка при чтении ответа (байт) |
| `timeout` | `30` | Таймаут запроса в секундах |
| `sleep` | `0.5` | Пауза между запросами в секундах |

## 🛠️ Как это работает

1. Создается HTTP-сессия с заголовком `User-Agent` (для обхода блокировок ботов)
2. Последовательно выполняется `num_requests` GET-запросов с `stream=True`
3. Для каждого запроса:
   - Засекается время начала
   - Ответ читается чанками по 8 КБ
   - Подсчитывается общий размер загруженных данных
   - Вычисляется скорость: `(байты × 8) / время / 1024 / 1024` → Мбит/с
4. После всех запросов вычисляются средние значения
5. Результаты печатаются в консоль

## ⚠️ Возможные проблемы

### Ошибка 403 Forbidden
Сервер блокирует запросы без `User-Agent`. Скрипт уже использует заголовок браузера, но некоторые серверы все равно блокируют. Попробуйте другой URL.

### SSL ошибки
Если возникает `SSLError`, используйте HTTP вместо HTTPS:
```bash
python speed_test.py http://speedtest.tele2.net/10MB.zip
```

### Таймаут
Если файл слишком большой или соединение медленное, увеличьте `timeout` в коде.

### Скрипт зависает
- Нажмите `Ctrl+C` для прерывания — скрипт корректно обработает это
- Проверьте соединение с сервером: `curl -I <URL>`
- Попробуйте другой URL из списка
